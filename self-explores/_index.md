# Project Context Index — skills-anthropic-official

Repository chứa các skill definitions chính thức của Anthropic cho Claude Code.
Folder này lưu trữ learnings, decisions, và context quan trọng của dự án nghiên cứu skill.

## Cấu trúc
- `decisions/` — Quyết định quan trọng
- `learnings/` — Kiến thức học được từ nghiên cứu skill
- `context/` — Bối cảnh dự án
- `daily/` — Nhật ký hàng ngày
- `tasks/` — Context cho từng task
- `leverage/` — Phân tích đòn bẩy bất đối xứng
- `samples/` — Mẫu công việc tái sử dụng

## Cách sử dụng
- `/viec ghi` — Ghi chép mới
- `/viec doc` — Đọc lại context
- `/viec xong` — Tự động hỏi learnings khi hoàn thành task
- `/viec review` — Tổng hợp vào daily log
